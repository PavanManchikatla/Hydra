# Brand assets

| File | Use | Provenance |
|---|---|---|
| `hydra-hero.png` | Banner / social card / README top | AI-generated image supplied by the repository owner; owner asserts usage rights per the generating tool's terms. |
| `hydra-logo.svg` | Small mark (favicon, avatar, inline icon) | Original work, project-owned. |

**Favicon / avatar guidance:** use `hydra-hero.png` for the banner and social card; use `hydra-logo.svg` for small sizes — the hero does **not** downscale legibly to 64px (five heads become mud). Small contexts (favicon, GitHub org avatar, ≤128px) must use the SVG mark.
